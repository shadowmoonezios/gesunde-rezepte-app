# Gesunde Rezepte App

Willkommen in der Gesunde Rezepte App! Diese Anwendung ist der perfekte Ort, um gesunde Rezepte zu finden, zu teilen und zu bewerten.

## Funktionen
- **Rezepte Durchsuchen:** Stöbern Sie durch eine Vielzahl von gesunden Rezepten nach Kategorien.
- **Rezepte Hochladen:** Teilen Sie Ihre eigenen gesunden Rezepte mit der Community.
- **Bewertungen:** Geben Sie Bewertungen für Rezepte ab, die Sie ausprobiert haben.
- **Benutzerkonto:** Erstellen Sie ein Benutzerkonto, um Ihre Lieblingsrezepte zu speichern und Ihre eigenen hochzuladen.

## Technologie-Stack
- **Frontend:** React, Redux
- **Backend:** Node.js, Express
- **Datenbank:** MongoDB
- **Auth:** JWT

## Installation
1. Klonen Sie das Repository: `git clone https://github.com/username/gesunde-rezepte-app`
2. Gehen Sie in das Verzeichnis: `cd gesunde-rezepte-app`
3. Installieren Sie die Abhängigkeiten: `npm install`
4. Starten Sie die Anwendung: `npm start`
